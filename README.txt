==== Extra Soft Downloader by SNeicer ====


==SYSTEM REQUIREMENTS==
Os: Windows 7/8/8.1/10
Ram: 5mb

==How to use==
1.Open app
2.Choose a category
3.Choose a app
4.Wait for download
5.Go to app directory and open Installed soft
6.Open needed category
7.Enjoy your new software :)
